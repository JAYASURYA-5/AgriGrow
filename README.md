# AgriGrow- Agricultural Management System

AgriGrow is a comprehensive agricultural management platform that combines multiple tools and features to help farmers optimize their farming practices, access market information, connect with communities, and leverage machine learning for disease prediction.

## 🌾 Features

- **Crop Disease Predictor**: ML-powered disease detection and recommendation system
- **Market Hub**: Real-time agricultural commodity pricing and market analysis
- **Farmer Community**: Connect with other farmers and share knowledge
- **Intercropping Assistant**: Optimize crop combinations for better yields
- **Weather Dashboard**: Real-time weather data and forecasts
- **LMS Portal**: Educational resources and farming videos
- **E-commerce Integration**: Buy and sell agricultural products
- **Economic Analysis**: Cost-benefit analysis tools for farming decisions

## 🚀 Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn package manager


## 📦 Available Scripts

- `npm run dev` - Start the development server with Vite
- `npm run build` - Build the project for production
- `npm run preview` - Preview the production build locally

## 🛠️ Technology Stack

### Frontend
- **React 18**: UI framework
- **Vite**: Fast build tool and dev server
- **React Router**: Client-side routing
- **Material-UI**: Component library
- **Tailwind CSS**: Utility-first CSS framework
- **Recharts**: Data visualization
- **Lucide React**: Icon library

### Backend/ML
- **Python**: For ML models and server
- **Node.js**: Runtime environment
