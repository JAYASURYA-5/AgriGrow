# AGR LMS Portal

A Learning Management System built with Next.js for agricultural education.

## Features

- User registration and login
- Browse available courses
- Course details and lessons
- Responsive design with Tailwind CSS

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `app/` - Next.js app router pages
- `components/` - Reusable components (to be added)
- `public/` - Static assets

## Technologies Used

- Next.js 16
- React
- TypeScript
- Tailwind CSS
- ESLint

## Contributing

Feel free to contribute to the project by adding more features like user authentication, database integration, and additional courses.
